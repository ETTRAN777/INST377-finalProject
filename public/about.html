Job Junction

About Job Junction

Job Junction is a modern web platform designed to help users discover, search, and bookmark IT jobs with ease. Our mission is to connect job seekers with employers and empower you to build your career in technology.

Developers

  • Ethan Tran
  • Chris Xu

Technologies Used

  • HTML5, CSS3 (with Lexend and Lexend Deca fonts)
  • JavaScript (Frontend and Backend)
  • Node.js & Express.js (Backend server)
  • Supabase (Authentication & Database)
  • Adzuna API (Job data)
  • GSAP, Swiper.js, Typed.js (UI/UX enhancements)
  • Google reCAPTCHA (Security)

Project Details

This site was created as a final project for INST377 at the University of Maryland. It demonstrates full-stack web development, API integration, and modern UI/UX design.
